# Definition of Done

The project is complete when:

- [ ] A public GitHub repository is available.
- [ ] The analysis runs from a clean R session.
- [ ] Dependencies can be restored using renv.
- [ ] The Eunomia OMOP sample database is downloaded at runtime.
- [ ] No raw database or patient-level export is committed.
- [ ] Cohort definitions are implemented in SQL.
- [ ] Study parameters are stored in configuration files.
- [ ] Propensity-score overlap is evaluated.
- [ ] Covariate balance is evaluated before and after adjustment.
- [ ] The primary effect estimate includes a confidence interval.
- [ ] Sex and age subgroup analyses are reported.
- [ ] At least three sensitivity analyses are reported.
- [ ] Tests, linting, and report rendering pass in GitHub Actions.
- [ ] The limitations clearly state that Eunomia is demonstration data.
- [ ] A tagged release v1.0.0 exists.
